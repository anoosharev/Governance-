# Governance-

Governance repo for Lydia/ZC code base 

Internal Record: Open Source License Election

Prepared by: Anoosh Arevshatian, CPO. 

Reviewed by: Mehdi Kennani, Head of Architecture; Lead Principal Engineer 

Scope: Zodia Custody Limited codebase, release 5.2.0.0

Purpose
This note records the Company's formal election of license terms for two open source components that are offered under multiple licenses on a disjunctive basis. Where a component is offered under more than one license, the user is entitled to choose which of those licenses to rely on. This note documents the choice the Company has made and relies upon.

Components and election
Logback (logback-core, logback-classic, v1.5.17), offered under EPL-1.0 or LGPL-2.1. The Company elects to use this component under the EPL-1.0 license.
JNR-POSIX (com.github.jnr:jnr-posix, v3.1.20), offered under EPL-1.0, GPL-2.0 or LGPL-2.1. The Company elects to use this component under the EPL-1.0 license and does not exercise the GPL-2.0 option.

Basis
Both components are consumed as unmodified upstream libraries, called through their public interfaces and not forked or source-modified. The EPL-1.0 license carries no obligation to disclose or make available the Company's proprietary source code arising from this use. The GPL-2.0 option listed for JNR-POSIX is one of several alternatives and is not relied upon.

Status: This election applies from the date above and remains in effect unless and until formally revised in writing.
